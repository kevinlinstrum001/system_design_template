# Documentation

This folder stores the repository-wide lessons, principles, vocabulary, diagrams, and explanatory material for the **System Design Template** project.

Its purpose is to preserve what the project teaches, not only what the project builds.

## Purpose

Use this folder for documentation that applies across the repository.

Typical contents may include:

- system-design lessons;
- vocabulary definitions;
- guiding principles;
- workflow explanations;
- architecture notes;
- design methods;
- comparison documents;
- decision records;
- diagrams;
- project-wide references;
- summaries of lessons learned across several examples.
