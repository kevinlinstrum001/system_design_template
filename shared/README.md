# Shared

This folder stores reusable code, styles, schemas, components, and supporting files that may be used by more than one example or application in the **System Design Template** repository.

The purpose of `shared/` is to reduce duplication while keeping common project elements organized and inspectable.

## Purpose

Use this folder for materials that are designed to be reused across multiple examples.

Typical contents may include:

- shared CSS;
- shared JavaScript;
- reusable HTML fragments;
- common JSON schemas;
- helper functions;
- interface components;
- shared configuration patterns;
- utility scripts;
- templates used by more than one application.
