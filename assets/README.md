# Shared Assets

This folder stores visual and media files used across the entire **System Design Template** repository.

It belongs at the repository root because its contents are not tied to one specific example. Assets that belong only to a single app or experiment should stay inside that example’s own folder.

## Purpose

Use this folder for shared materials such as:

- diagrams;
- infographics;
- screenshots;
- icons;
- logos;
- illustrations;
- reusable interface graphics;
- printable reference images;
- other media used by multiple pages or documents.
