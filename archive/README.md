# Archive

This folder stores project materials that are no longer active but remain useful for history, comparison, recovery, or reference.

The archive exists so the repository can preserve earlier work without allowing outdated files to clutter the active project structure.

## Purpose

Use this folder for materials such as:

- retired examples;
- superseded documents;
- old data structures;
- discontinued prototypes;
- replaced diagrams;
- earlier versions of static pages;
- experiments that failed but produced useful lessons;
- files kept for comparison or recovery.

## What belongs here

Move something into `archive/` when it is no longer part of the current working system but still has value.

Examples:

```text
archive/
├── examples/
├── documents/
├── diagrams/
├── data/
└── README.md
